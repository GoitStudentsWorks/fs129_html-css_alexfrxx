# Handmade Jewelry

This project is a modern landing page for a handmade jewelry brand, designed to
showcase unique, handcrafted pieces and create an engaging user experience.

The goal of the project is to combine aesthetic design with clean, responsive
front-end development, ensuring the website looks great across all devices while
highlighting the brand’s identity and products.

---

## 🔗 Live Demo

👉 https://alexfrxx.github.io/handmade-jewelry-landing/

---

## 🛠 Tech Stack

HTML – semantic and accessible page structure CSS – styling, layout, and
responsive design JavaScript – interactivity and dynamic behavior Vite – fast
development environment and build tool Figma – UI/UX design and prototyping Git
& GitHub – version control and team collaboration

---

## 🎯 Features

Responsive design (mobile-first approach) Smooth navigation and user-friendly
layout Modern UI based on Figma design Interactive elements powered by
JavaScript Optimized performance with Vite
